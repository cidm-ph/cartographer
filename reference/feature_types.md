# List known feature types

Each feature type corresponds to map data that has been registered.

## Usage

``` r
feature_types()
```

## Value

Character vector of registered feature types.

## See also

[`register_map()`](https://cidm-ph.github.io/cartographer/reference/register_map.md)

## Examples

``` r
feature_types()
#>  [1] "maps.italy"                    "rnaturalearth.countries_hires"
#>  [3] "maps.lakes"                    "rnaturalearth.countries"      
#>  [5] "rnaturalearth.australia"       "maps.nz"                      
#>  [7] "maps.world"                    "maps.state"                   
#>  [9] "sf.nc"                         "maps.france"                  
```
